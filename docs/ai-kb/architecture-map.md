# Architecture Map

## Top-Level Modules

- `backend/`: backend service implementation
- `frontend/`: frontend interface and assets
- `deploy/`: deployment scripts and runtime packaging
- `tools/`: utility scripts and supporting tooling
- `docs/`: project documentation and AI knowledge base

## Navigation Hints

1. Start with `README.md` and `DEV_GUIDE.md` for project entry points.
2. Map requested changes to one primary module first.
3. Search by behavior keyword before editing.
