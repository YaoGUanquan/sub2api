# AI Knowledge Base

This folder stores compact, durable project memory for future coding sessions.

## Files

- `project-context.md`: stable project identity and constraints
- `architecture-map.md`: high-level module map
- `decision-log.md`: notable technical decisions
- `troubleshooting-log.md`: incidents and fixes
- `session-notes.md`: short execution notes by date
- `backlog.md`: next tasks and follow-ups

## Update Rules

1. Keep each entry short and factual.
2. Add date in `YYYY-MM-DD` format.
3. Link related code paths when possible.
4. Capture rationale, not only outcomes.
