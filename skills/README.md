# Skills Index

Project-local skills used by `AGENTS.md` for intent-based routing.

## Available Skills

- `project-intake-planning`: clarify scope and produce execution plans
- `go-backend-implementation`: implement backend changes safely
- `bug-triage-debugging`: reproduce and fix issues with evidence
- `knowledge-base-maintenance`: maintain docs and AI knowledge base

Each skill is self-contained and includes:

- `SKILL.md`: trigger description and workflow
- `agents/openai.yaml`: UI metadata
- `references/`: focused checklists or playbooks
